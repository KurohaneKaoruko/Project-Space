# 一些小项目的网站

## 项目介绍

这是一个展示各种小项目的网站，包括一些有趣的小游戏、工具和应用程序。

## 技术栈

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS + DaisyUI
- MongoDB

## 项目列表

### 2048 游戏

经典的 2048 数字合并游戏，支持以下功能：

- 多种棋盘大小 (4x4 ~ 7x7)
- 撤销功能
- 本地存档
- 键盘和触屏操作
- AI 自动游玩（三种模式）

#### AI 玩家模式

| 模式 | 算法 | 特点 |
|------|------|------|
| 快速 | 固定优先级 | 响应迅速，简单策略 |
| 均衡 | Minimax | 平衡速度与效果 |
| 最优 | Expectimax | 深度搜索，效果最佳 |

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
